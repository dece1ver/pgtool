
# pgtool

Утилита командной строки для сканирования архива управляющих программ станков с ЧПУ и экспорта информации об инструментах в JSON.

## Что делает

Обходит директорию с архивом программ, распознаёт иерархию **станок → группа деталей → [подгруппа] → деталь → [подпапка] → программа**, извлекает список инструментов из каждой программы и сохраняет результат в JSON. В процессе работы выводит прогресс по каждому станку параллельно.
Директории в квадратных скобках опциональны.
## Структура архива

Ожидаемая структура директорий:

```
<АРХИВ>/
└── <Станок>/
    └── <Группа деталей>/
        └── <Деталь>/
            └── программа.pbg / .maz / .h / .mpf / .nc / ...
```
Папки с деталями распознаются по префиксам обозначений (пока хардкод) (например `АР`, `АРМ`, `10-`, `25-` и др.).

## Поддерживаемые форматы

| Расширение | Формат      | Статус |
|------------|-------------|--------|
| `.pbg`     | Mazatrol    | ✔      |
| `.maz`     | Mazatrol    | ✔      |
| `.h`       | Heidenhain  | потом  |
| `.mpf`     | Sinumerik   | потом  |
| прочие     | G-code      | ✔      |

Исполняемые, графические, архивные и офисные файлы автоматически пропускаются.

## Использование

```
pgtool [ФЛАГИ] <АРХИВ> [--output <ПУТЬ>]
```

### Аргументы

| Аргумент              | Описание                                                    |
|-----------------------|-------------------------------------------------------------|
| `<АРХИВ>`             | Путь к корневой директории с программами для станков            |
| `-o, --output <ПУТЬ>` | Путь к выходному JSON-файлу (по умолчанию: `output.json`)   |
| `-f, --full`          | Включить полные данные (детали и программы) в JSON          |
| `-h, --help`          | Справка                                                     |
| `-V, --version`       | Версия                                                      |

### Примеры

```bash
# Минимальный запуск — сводка по станкам в output.json
pgtool D:\Programs

# Указать выходной файл
pgtool D:\Programs --output report.json

# Полный вывод с деталями и программами
pgtool D:\Programs -fo C:\reports\tools.json
```

## Формат вывода JSON

### Без флага `--full` (сводка)

```json
[
  {
    "name": "Станок 1",
    "part_groups": [
      { "name": "Группа АР", "avg_tools_count": 12 }
    ]
  }
]
```

### С флагом `--full`

Дополнительно включает список деталей, программ и инструментов каждой программы.

```json
[
  {
    "name": "Станок 1",
    "part_groups": [
      {
        "name": "Группа АР",
        "avg_tools_count": 12,
        "parts": [
          {
            "name": "АР-001",
            "programs": [
              {
                "name": "prog.pbg",
                "tools": [
                  { "number": 10, "corrector": "A" },
                  { "number": 11, "corrector": "B" }
                ]
              }
            ]
          }
        ]
      }
    ]
  }
]
```
